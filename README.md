# blur_images

A simple image processor that applies the blurry background effect to images

## Setup

[setup.bat](setup.bat)

- It creates a python virtual environment
- It installs the dependencies

## Execute

[execute.bat](execute.bat)

- It runs the program

## Dist

[dist.bat](dist.bat)

- It creates the executable
